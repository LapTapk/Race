\doxysubsection{Detailed Description}
Класс, отвечающий за проверку нахождения машины в пределах дороги. Проверка осуществляется с помощью проверки пересечения отрезков границ дороги и создаваемого данной сущностью отрезка. Последний создается от позиции родительского объекта до машины.
